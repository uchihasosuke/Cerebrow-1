# CEREBROW - Browser-Based AI Agent
# Comprehensive Feature Documentation

## PROJECT OVERVIEW
Cerebrow is a sophisticated browser-based AI agent built on Gradio that provides intelligent web automation capabilities. It leverages multiple Large Language Models (LLMs) to understand and execute complex web tasks autonomously while maintaining a user-friendly interface.

## CORE FEATURES

### 1. MULTI-LLM SUPPORT
- **OpenAI Models**: GPT-4o, GPT-4, GPT-3.5-turbo, o3-mini
- **Google Models**: Gemini 2.5 Flash Lite, Gemini 2.5 Flash, Gemini 2.5 Pro, Gemini 2.0 Flash, Gemini 2.0 Flash Lite
- **Anthropic Models**: Claude 3.5 Sonnet (2024-10-22, 2024-06-20), Claude 3 Opus (2024-02-29)
- **DeepSeek Models**: DeepSeek Chat, DeepSeek Reasoner
- **Azure OpenAI**: Full OpenAI model compatibility through Azure
- **Ollama Models**: Qwen2.5 (7B, 14B, 32B), Qwen2.5-Coder (14B, 32B), Llama2 (7B), DeepSeek-R1 (14B, 32B)
- **Mistral Models**: Pixtral Large, Mistral Large, Mistral Small, Ministral 8B
- **Alibaba Models**: Qwen Plus, Qwen Max, Qwen VL Max, Qwen VL Plus, Qwen Turbo, Qwen Long
- **Moonshot Models**: Moonshot V1 (32K, 8K Vision Preview)
- **Unbound AI Models**: Gemini 2.0 Flash, GPT-4o Mini, GPT-4o, GPT-4.5 Preview
- **Grok Models**: Grok-3, Grok-3 Fast, Grok-3 Mini, Grok-3 Mini Fast, Grok-2 Vision, Grok-2 Image, Grok-2
- **SiliconFlow Models**: DeepSeek R1/V3, Qwen models, InternLM models, ChatGLM models
- **IBM Models**: Granite Vision 3.1, Llama models
- **ModelScope Models**: Comprehensive Qwen model collection

## TECHNICAL SPECIFICATIONS

### System Requirements
- **Python**: 3.11 or higher
- **Memory**: Minimum 8GB RAM (16GB recommended)
- **Storage**: 2GB available disk space
- **Network**: Internet connection for LLM APIs

### Supported Platforms
- **Operating Systems**: Windows, macOS, Linux
- **Browsers**: Chromium, Firefox, WebKit
- **Architectures**: x86_64, ARM64

### Performance Characteristics
- **Response Time**: Sub-second to few seconds depending on task complexity
- **Concurrent Tasks**: Up to 10 parallel browser operations
- **Memory Usage**: 2-8GB depending on browser instances
- **CPU Usage**: Moderate to high during active automation
